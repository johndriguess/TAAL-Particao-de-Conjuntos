# Taal - Problema da Partição de Conjuntos

## Problema de Partição de Conjuntos

### Descrição
Dado um conjunto de números inteiros positivos, determine se é possível dividi-lo em dois subconjuntos com soma igual.

### Entrada
```text
n
a1 a2 a3 ... an
```

### Saída
```text
YES
k
subset1
subset2
```

Ou

```text
NO
```
